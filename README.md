## Description

Ruby bindings to FAM API via FFI.

## Features

Right now it's a pre-alpha hack which I hope to develop into something usable.

## Examples

    require 'rb-fam'

## Requirements

FAM or Gamin SHOULD be installed as an API provider.

## Install

    $ gem install rb-fam
